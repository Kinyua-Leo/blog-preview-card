## Overview
 This project is my solution to the Blog Preview Card Challenge from Frontend Mentor. Recreating the provided design as accurately as possible while writing clean,semantic HTML and maintainable CSS was the goal.

### The challenge

Users should be able to:

- See hover and focus states for all interactive elements on the page
- View the optimal layout on different screen sizes

## My process

I approached the project by first analyzing the design and identifying its individual components.I then used semantic HTML to build the layout before applying CSS for styling,typography,colors & responsiveness.Finally I tested the project across different screen sizes.

### Built with

- Semantic HTML5 
- CSS3
- Flexbox
- CSS Grid
- Mobile-first workflow
- Visual Studio Code 


### What I learned

I reinforced several important concepts,including:
- Writing semantics and accessible HTML
- Working with typography colors & shadows.
- Creating layouts with flexbox.
- Managing images and relative paths. 
- using dev tools to test responsiveness.


### Continued development

I plan to focus on:
- Writing cleaner and more scalable CSS.
- Improving responsive design techniques.
- Building layouts more efficiently.
- Following modern accessibility best practices.
- Sharpening my debugging skills and use of dev tools.
